# Project Initialization Checklist

- [ ] Define problem clearly
- [ ] Define target user
- [ ] Define MVP scope
- [ ] Define success metric
- [ ] Write first LinkedIn post draft
- [ ] Define 90-day goal
- [ ] Set up domain (if needed)
- [ ] Create first issue in GitHub

# Commit rules

Example:

```bash
feature: build function to extract data from yfinance
fix: correct async bug in data fetcher
refactor: simplify auth middleware
docs: update roadmap for MVP
chore: update dependencies
```
